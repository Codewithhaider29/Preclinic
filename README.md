# 🏥 Preclinic – Medical Admin Dashboard Template

**Preclinic** is a fully responsive, lightweight, and highly customizable **Medical Admin Dashboard Template** built for hospitals, clinics, and healthcare platforms. Powered by **Bootstrap 5+, HTML5, CSS3**, and modern JavaScript frameworks, Preclinic offers versions ready to use in **Laravel, React, Angular, Vue.js, and Node.js (EJS)** environments.

---


## ⚙️ Available Tech Stacks

Preclinic is available in the following technologies:

- **Bootstrap 5.3.3**
- **React 19**
- **Angular 19**
- **Vue 3**
- **Laravel 11**
- **Node.js (EJS)**
